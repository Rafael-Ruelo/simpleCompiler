# simpleCompiler
Final Project Compiler how to use:

1. Run .main file
2. GUI will automatically load up
3. Keywords to use:
   - lalagyan = var (for declaring variables)
   - ilagay = input (user input)
   - kinalabasan = output (outputting values)
4. Sample code you can run (copy this directly):
   - lalagyan a
   - lalagyan b
   - lalagyan sum
  
   - ilagay a
   - ilagay b
  
   - sum = a + b
  
   - kinalabasan sum
  
5. Done!
